# Basic Calculator

A basic Calculator made with python for non-scientific calculations.
This is my first project, i made it to gain more experience on pytohn and get used to VSCode. It's nothing fancy, just a simple calculator, but it taught me alot.

## Features
- Add, Subtract, Multiply, Divide
- GUI interface
- Packaged into a Windows .exe with pyinstaller

## How to run 
Download the latest .exe from the Realease page

## Built with
- Python
- Tkinter
- Pyinstaller
